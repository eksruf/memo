git clone (깃허브코드 주소)
=다른 곳에서 작업한 후 깃허브에서 푸쉬한 모든 내용을 폴더형태로 내려받는다

git branch -r
=지정한 원격지에 대한 브런치를 보여준다

git branch -a -pc
=원격지에 대한 브런치 모두를 보여준다

git push (깃허브별칭) --delete (브런치명)
=원격지에 있는 브런치 삭제

깃허브에도 git bash처럼 여러 브런치로 파일을 관리할 수 있다.

원격지에 이미 브런치가 있다면 clone을 하고 브런치를 다시 만들 필요없이
checkout을 통해서 변경만 해주면 된다.
